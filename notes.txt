File structure:
    pieces.py: Contains the Piece class and all subclasses.
    board.py: Contains the Board class.
    game.py: Contains the Game class.
    graphics.py: Contains rendering functions for board and pieces.
    chess_logic.py: Contains functions that implement the rules of chess.
    event_handler.py: Contains functions that handle events.
    main.py: Contains the main function.
    tests.py: Contains unit tests for most functions in other modules.
    README.md: Contains the README file.
